# U.S. States Game (Python Turtle & Pandas)

A simple Python geography game where the player tries to guess all 50 U.S. states.

This project was built using Python, Turtle Graphics, Pandas, CSV files, and a graphical map of the United States.

---

## Features

* Guess all 50 U.S. states
* Displays correct state names on the map
* Tracks the player's progress
* Uses state coordinates from a CSV file
* Generates a list of missing states when exiting the game
* Interactive graphical interface with Turtle Graphics

---
